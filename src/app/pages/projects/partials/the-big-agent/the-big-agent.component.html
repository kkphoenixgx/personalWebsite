Logical Structure and Scalability of The BIG Agent

[cite_start]The BIG Agent proposes an architecture focused on decomposing the control of Unmanned Aerial Vehicles (UAVs)[cite: 2]. [cite_start]The system fragments the problem into two fundamental Multi-Agent Systems (MAS): one responsible for the physical interface and direct actuation of the flight hardware [cite: 8][cite_start], and a central node that processes route planning and tactical orchestration[cite: 9, 59]. [cite_start]The validation, executed in the Webots simulator using a DJI Mavic 2 Pro model [cite: 46, 52][cite_start], proves the viability of mapping logical commands to physical security operations[cite: 33].

The main pillar of the project is the strict decoupling of layers. [cite_start]Physical perception and motor control are isolated from external network communication, which operates over the ContextNet middleware[cite: 110, 115, 134]. By removing the low-level hardware syntax from the responsibilities of the BDI tactical layer, the system achieves the modularity necessary to ensure that updates or occasional failures do not corrupt the control architecture.

The modeling resolves the structural incompatibility between the discrete reasoning of the agents and the continuous physics of the environment. [cite_start]To handle the latency inherent in sending data, aerodynamic stability is managed by an internal Proportional-Derivative (PD) algorithm that forces the UAV to return to a stationary flight state (hovering) between the execution of each movement packet[cite: 95, 108]. This mechanics prioritizes the precision of the premise over fluidity, ensuring predictability.

Finally, the expansion capacity is intrinsic to the network topology. [cite_start]The communication architecture uses illocutions based on unique UUIDs, defining a rigorous standard for identifying and exchanging messages[cite: 136, 137]. [cite_start]This taxonomy builds the foundation necessary for the solution to move from unitary control and support scaling to swarms of UAVs[cite: 60], connecting the solid logic of MAS to practical execution at Edge AI frontiers.
