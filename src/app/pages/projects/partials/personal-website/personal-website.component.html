frontend-renderer.ts

WebGL, Física Dinâmica e Zero-GC

A interface dissolve a barreira entre UI e simulação. Utilizamos Three.js para renderizar Sistemas Solares com Raycasting nativo, e injetamos Matter.js para gerar uma lâmpada física sujeita à gravidade no DOM.

Para proteger a GPU, laços de renderização são geridos cirurgicamente por IntersectionObservers e componentes pesados nascem por Lazy Loading via ViewContainerRef, descarregando a memória (Zero Garbage Collection Thrashing) quando invisíveis.
